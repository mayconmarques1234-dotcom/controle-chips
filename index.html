🔒 Aviso de Privacidade (LGPD)

Este sistema coleta e armazena dados para fins de controle interno operacional:
• Nomes de usuários e licenciados — identificação de responsáveis
• Números de telefone — registro de operações de recarga/ativação
• ICCIDs — rastreamento de chips físicos

Os dados ficam armazenados no Google Sheets da organização e no navegador local (localStorage). Nenhum dado é compartilhado com terceiros além dos serviços Google utilizados. O acesso é restrito por senha e destinado exclusivamente ao uso interno da equipe iGreen.

Instruções de configuração

1. Crie uma planilha com abas lancamentos e estoque
2. Em console.cloud.google.com → crie projeto → ative Google Sheets API
3. Crie uma Chave de API e restrinja ao domínio do GitHub Pages
4. Cole o ID e a chave acima → "Salvar e testar"
5. Compartilhe a planilha como "Qualquer pessoa com o link pode editar"

Upload de planilha de vendas
A planilha deve ter 3 abas obrigatórias: licença, chip avulso e Dom.
Regras de contagem:
• VOUCHER + EXPERT ou EXPERT+TELECOM + (Integral ou Boleto≠1/4) = 2 chips
• PAGO + EXPERT+TELECOM + (Integral ou Boleto≠1/4) = 2 chips
• PAGO + TELECOM + (Integral ou Boleto≠1/4) = 10 chips
• RENOVAÇÃO e RENOVAÇÃO VOUCHER = não contabiliza
• Chip avulso só é contado se o ID_TRANSACAO bater com algum payer_name da aba Dom
